# Express Backend Template

API was designed using useEffect and async functions with direct paths to requested pages from the frontend to the backend server. Applied try catch for error handling.

This project is built to help manage projects and task with filters for pending and completion. You can also delete and edit these task. The dependencies listed below were installed to assist with code as well.


## Dependencies

- Express
- MongoDB/Mongoose
- Dotenv
- Morgan
- Cors
- Helmet


## Dev Dependencies

- Nodemon
